# 기초 CLI 명령어

> CLI : Command Line Interface
>
> GUI 환경인 윈도우와는 다른 인터페이스를 지닌다.

* ls : 목록

  ```bash
  $ ls
  1.txt
  ```

* touch : 파일 만들기

  ```bash
  $ touch <파일명>
  $ touch README.md
  ```

* pwd : 현재 작업 디렉토리 출력 (print working directory)

  ```bash
  $ pwd
  /c/Users/lec/Desktop/실습1

* mkdir : 디렉토리 만들기 (make directory)

  ```bash
  $ mkdir test
  ```

* cd : 디렉토리 이동 (change directory)

  ```bash
  $ cd test
  ~/Desktop/실습1/test $ cd ..
  ~/Desktop/실습1 $ 
  ```

  * `..` : 상위 디렉토리
  * `.` : 현재 디렉토리
